## Execution Steps
## 执行步骤
1. Set measurable NFR metrics for performance, stability, security, and maintainability.
1. 制定性能、稳定性、安全性、可维护性等可量化NFR指标。
2. Define capacity baseline as 20 servers and 8 processes per server under stable mode.
2. 将稳定模式容量基线定义为20台服务器、每台8进程。
3. Define controlled upward redundancy mode with at least +20% burst capacity.
3. 定义受控向上冗余模式，至少支持+20%突发能力。
4. Bind each metric to verification method and evidence output.
4. 为每项指标绑定验证方法与证据输出。
